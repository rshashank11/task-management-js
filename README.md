# Sortable Drag & Drop Task Management
## Functionalities:
1. Three sections:
   - To do
   - In progress
   - Completed
2. Add Task:
   - Add Task button when clicked changes UI to
   - Enter task & description and Save button
3. Sortable drag and drop feature
